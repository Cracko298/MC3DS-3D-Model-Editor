# MC3DS-Model-Editor
Edit Minecraft 3DS Edition Models in 3D using this GUI.



## Requirements
### Install Requirements from PowerShell:
```
pip install -r requirements.txt
```

### Modules:
```
matplotlib==3.9.2
numpy==1.24.4

customtkinter==5.2.2

```
